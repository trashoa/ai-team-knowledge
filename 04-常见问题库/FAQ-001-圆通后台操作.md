# 圆通后台自动化操作常见问题

## 系统信息

- **网址**: https://kh.yto.net.cn/
- **登录方式**: 账号密码 + 图形验证码（计算题）
- **常用功能**: 运单打印 → 自定义打印

## 常见问题

### 1. 图形验证码识别

**问题**: 圆通后台使用计算题验证码（如 "3 + 5 = ?"）

**解决方案**:
```bash
# 使用 agent-browser 截图后人工识别
agent-browser screenshot --selector ".captcha-img"

# 或接入 OCR 服务自动识别（待开发）
```

### 2. 地址一键解析失效

**问题**: 页面"一键解析"按钮点击无反应

**解决方案**:
```javascript
// 手动拆分地址
// 原始: "甘肃平凉市崆峒区东关街道沙港市场占峰牛肉面转4308"
// 拆分:
//   省份: 甘肃省
//   城市: 平凉市
//   区县: 崆峒区
//   详细: 东关街道沙港市场占峰牛肉面转4308
```

### 3. 下拉菜单选择不稳定

**问题**: agent-browser 点击下拉菜单项失败

**解决方案**:
```bash
# 使用 mouse move + mouse down + mouse up 组合
agent-browser click --selector "select" --action "mouse"
agent-browser keyboard --key "ArrowDown"
agent-browser keyboard --key "Enter"
```

### 4. 商品信息未填写

**问题**: 打单后商品信息为空

**解决方案**:
```
路径: 自定义打印 → 已打印 → 更多操作 → 修改商品信息
格式: 货号-尺码 出货数量
示例: 355152-140 42 出货
```

## 操作技巧

1. **登录状态保存**: `/tmp/yto_auth.json`
2. **截图调试**: `/tmp/yto_*.png`
3. **Tab 键切换**: 表单填写更稳定
4. **键盘输入**: `keyboard type` 比 `type` 更可靠
