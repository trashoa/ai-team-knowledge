---

## 📖 简介

知识库是给 **AI 用的**，不是给人看的。

- AI 审查时自动检索相关知识
- 把知识注入提示词，提升审查质量
- 新 AI 成员加入后立即就能用

---

## 📁 目录结构

```
知识库/
├── 01-安全漏洞库/      # 安全相关问题
├── 02-审查规则库/      # 代码审查规则
├── 03-修复方案库/      # 常见修复方案
├── 04-常见问题库/      # 常见问题解答
└── FORMAT_STANDARD.md  # 格式标准
```

---

## 📝 格式标准

详见：[FORMAT_STANDARD.md](./FORMAT_STANDARD.md)

### 快速开始

每个知识条目包含：

1. **YAML 头**：id、分类、严重程度、标签等
2. **问题描述**：清晰描述问题
3. **错误示例**：展示错误代码
4. **正确做法**：展示正确代码
5. **检测方法**：如何检测
6. **修复方案**：如何修复

---

## 🔄 入库流程

1. 大黑手审查发现问题
2. 在 Issue 评论打标签 `[建议入库]`
3. 光枢每天检查并整理
4. 提交到知识库
